# AAC Communication System - Flutter Frontend

Aplicație Flutter Web pentru sistemul de comunicare AAC.

## Instalare

1. Asigură-te că ai Flutter instalat:
```bash
flutter --version
```

2. Instalează dependențele:
```bash
flutter pub get
```

3. Rulează aplicația:
```bash
flutter run -d chrome
```

## Configurare

Asigură-te că backend-ul rulează pe `http://localhost:8000` înainte de a porni aplicația Flutter.

Pentru a schimba URL-ul backend-ului, editează `lib/services/api_service.dart` și modifică constanta `baseUrl`.
